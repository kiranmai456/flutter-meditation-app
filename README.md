# 🚀Flutter Meditation App

A modern wellness application built with **Flutter** and **Dart** that provides a calm and intuitive experience for exploring meditation sessions, mindfulness content, and relaxation exercises. The project follows a **feature-based architecture** with reusable widgets, centralized routing, and a responsive user interface to ensure scalability and maintainability.

## 📂 Project Structure

```text
lib/
│
├── core/
├── features/
├── routes/
├── shared/
│   └── widgets/
└── main.dart
```

### 📁 Folder Description

* **core/** – Shared resources such as themes, colors, constants, utilities, and common services.
* **features/** – Organizes the application by feature, with each feature containing its own UI, logic, and models.
* **routes/** – Centralized route definitions for navigation.
* **shared/widgets/** – Reusable UI components used throughout the application.
* **main.dart** – Entry point of the application that initializes the app, theme, and initial screen.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/kiranmai456/flutter-meditation-app.git
```

### Navigate to the project

```bash
cd flutter-meditation-app
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```
